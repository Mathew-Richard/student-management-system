🎓 Student Management System
A web app to manage student records and calculate grades.
✨ Features
Add Students: Save names and 6-digit IDs.
Auto-Calculate: Averages Year 1, 2, and 3 CGPA.
Pass/Fail: Automatically marks students as "Pass" if their average is 5.0 or higher.
Error Check: Prevents entering a CGPA higher than 10.0.
🛠️ Tools Used
Python (The "Brain")
Flask (The Web Framework)
HTML/CSS (The Design)
🚀 How to Run
Open your terminal or CMD.
Type pip install flask and press Enter.
Type python app.py and press Enter.
Open your browser to: http://127.0.0.1:5000
